# Nightwing

Nightwing is a basic forensic multi-agent scaffold for triage, analysis, validation, and reporting.

## Quick Start

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies:
   - `python -m venv .venv`
   - `.venv\\Scripts\\activate`
   - `pip install -r requirements.txt`
3. Run CLI:
   - `python -m nightwing.main --help`

## Docker

Run local supporting services:

`docker compose up -d`
